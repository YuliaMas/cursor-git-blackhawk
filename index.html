Git command

Command Description
git config --global user.name Yulia вводимо своє ім'я для логування
git config --global user.email yulia.masechko@gmail.com вводимо свій е-мейл
git init ініціалізуємо репозиторій
git status перевіряємо, що є в нашому репозиторії
git add додаємо файл в репозиторій
git commit закомічуємо файл
git log показує історію, які є коміти,показує файли і які там відбулись зміни
git checkout вказуємо ще номер id, який копіюємо з попередньої команди git log, можна повернутись в часі і отримати початкові файли та так само повернутись назад до теперішньої версії файлів
git branch створення нової гілки репозиторію, за замовчуванням вона називається master
git push вигрузити всі локальні зміни у віддалений репозиторій
git pull стягнути всі зміни з віддаленого репозиторію собі на локальний
git merge
git clone
.gitignore файл
git revert
git reset
git rebase
git cherry-pick
git stash
